# Communications

Outreach + customer-facing email and message templates. Plain-prose
versions of pitches; copy-paste into Gmail / a CRM / a one-off send,
edit the salutation and signature, and go.

Voice rules:

- NZ English (colour, organisation, etc.). No em-dashes.
- Trademark the tournament name: **FIFA World Cup 2026&trade;** on
  first reference, then "the World Cup" or "the tournament".
- Always disclose: Tournamental is independent and not affiliated
  with FIFA.
- Lead with the tournament dates and the opening match so the reader
  knows we're talking about June 11 2026, Mexico vs South Africa,
  Estadio Azteca.
